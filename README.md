## 🚀 Deploy on Vercel

### One-click deploy:
1. Push this repo to GitHub (already done)
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo `nakulsinghkalyanwat/clairmaunt`
4. Vercel auto-detects static site — just click **Deploy**
5. Your site is live in ~60 seconds ✅

### No build step needed — this is a static HTML site.
